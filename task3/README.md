# Лабораторная №3

## Описание

Программа предназначена для нахождения ядра противоречий между двумя кластерными ранжировками на основе метода согласования экспертных оценок. 

Программа выполняет следующие вычисления:

**Анализ кластерных ранжировок** — обрабатывает две ранжировки, представленные в JSON-формате, где:
- Отдельные элементы представляют объекты с уникальным порядком
- Списки элементов в квадратных скобках представляют кластеры неразличимых объектов

**Построение матриц предпочтений** — для каждой ранжировки строится бинарная матрица, где элемент `[i][j] = 1`, если объект i не хуже объекта j (лучше или неразличим).

**Выявление противоречий** — находит пары объектов, для которых эксперты имеют противоположные мнения:
- Если один эксперт считает объект A лучше B, а другой — B лучше A
- Противоречивые пары образуют ядро противоречий

**Формирование результата** — возвращает JSON-строку, содержащую список пар объектов, составляющих ядро противоречий.

## Теоретическая основа

Метод основан на теории кластерных ранжировок и включает:
1. Перевод ранжировок в матричную форму
2. Поэлементное сравнение матриц предпочтений
3. Выявление пар с противоположными отношениями
4. Формирование ядра противоречий для последующего согласования экспертных мнений
